
# 안녕하세요, 백엔드 개발자 이태현입니다 👋

헬스케어·CRM 도메인에서 4년 이상 백엔드 개발을 해왔습니다.  
NestJS · TypeScript · ASP.NET Core 기반으로 설계부터 배포·운영까지 전 과정을 수행해왔습니다.  
레거시 현대화, 대용량 메시징 최적화, CI/CD 구축, 모니터링 환경 구성에 강점이 있습니다.

## 소개

- 헬스케어 / EMR / CRM 도메인 백엔드 개발 경험
- NestJS, TypeScript, ASP.NET Core, C# 기반 서비스 설계 및 운영
- Docker, GitHub Actions, Nginx 기반 배포 및 운영 자동화 경험
- Redis, RabbitMQ, MySQL, MSSQL, Oracle 기반 데이터 처리 및 성능 최적화 경험
- 실제 운영 중인 사이드 프로젝트와 기술 블로그를 꾸준히 관리 중
