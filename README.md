## 👋 소개

성장과 경험을 즐기는 **백엔드 개발자 이태현**입니다.

C# 기반 응용 프로그램 개발과 **ASP.NET Core**, **NestJS**를 활용한 백엔드 시스템 구축에 관심이 많습니다.

다양한 도메인의 실무 경험을 바탕으로 기술을 활용한 **실용적인 문제 해결**을 지향합니다.
